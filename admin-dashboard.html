Homepage Sections

Add a block to the homepage custom area. It can optionally appear in the public header as an anchor link.

Header Pages

Create pages that open through page.html?slug=your-page. Turn on the header option to show them beside Blog and Premium.

Articles

The homepage article section and blog.html read from this list. Default real articles are loaded automatically if the list is empty.

Stripe Checkout

Premium buttons are wired to these settings. Add real Stripe price IDs and a backend checkout endpoint to make subscriptions work in production.

Static HTML cannot create Stripe Checkout Sessions by itself. Your backend endpoint must create a session with the selected priceId and return { "url": "https://checkout.stripe.com/..." }.

Supabase Sync

Use a real Supabase project URL and anon key to let content stay shared across browsers and devices instead of remaining inside one local browser profile.

This sync manager only works once the actual Supabase project URL and anon key are saved and the table site_content exists with the policy script from supabase/schema.sql.
